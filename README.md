# Arena — Plataforma de Customer Operations Omnicanal (Demo)

Demo interactiva del journey omnicanal completo: bot conversacional → caso → consola del agente → dashboard operacional → callback outbound.

## Version

`0.2.0` — Outbound callback via Vapi/Exitus

## Implemented functionality

### Core
- [x] Channel selector (WhatsApp, SMS, Web, Facebook, Instagram, Voz) with per-profile branding
- [x] Text bot (Claude Sonnet 4) with intent detection, data capture, and auto case creation
- [x] Voice bot (OpenAI Realtime API) with WebRTC and live transcription
- [x] Agent inbox with filters by status, priority, SLA risk
- [x] Case detail with full timeline, reply, AI response suggestions
- [x] Operational + business dashboards (Recharts, SSE real-time updates)
- [x] Alerts view (SLA at-risk, overloaded agents)
- [x] Interactive guided tour
- [x] Command Center `/demo` page
- [x] 6 swappable company profiles (Megacable, JLL, MedTech, Banco, Ecommerce, Telco)
- [x] Demo seeder (300 cases)

### Outbound callback (new in 0.2.0 — telco profile only)
- [x] Text bot detects callback intent ("llámame", "prefiero hablar") and captures phone
- [x] Floating "📞 Prefiero que me llamen" button in the customer chat
- [x] Chat summary generated by Claude (≤150 words, Spanish)
- [x] Case auto-created with `canalOrigen: "voz"` + two system interactions
- [x] POST to Vapi/Exitus outbound endpoint with hardcoded telco payload
- [x] Graceful fallback when Vapi fails (case created, UI shows error message)

## Stack

Next.js 16 (App Router) + Tailwind 4 + Prisma + SQLite + Claude API (Anthropic) + OpenAI Realtime (voice) + Vapi/Exitus (outbound callback).

## How to run locally

```bash
npm install
cp .env.example .env
# Edit .env and set DATABASE_URL (required)
# Optional: ANTHROPIC_API_KEY, OPENAI_API_KEY, VAPI_OUTBOUND_URL
npx prisma db push
npm run dev
```

Open http://localhost:3000.

### Testing the callback flow

1. Home → select **Telco (Conecta Telecom)**
2. Go to `/cliente` → pick WhatsApp (or any channel)
3. Write "hola", then "prefiero que me llamen al 5512345678"
4. The bot confirms; click **Sí** or just write "confirmo"
5. Frontend POSTs to `/api/callback` → case created + Vapi fired

If `VAPI_OUTBOUND_URL` is not set, the endpoint defaults to the Exitus dev URL. For the callback button to appear, the active profile must be `telco-ejemplo`.

## How to run tests

```bash
make test      # or: npm test
```

Runs Vitest against `src/**/__tests__/*.test.ts`. Current coverage:
- `src/lib/__tests__/vapi.test.ts` — payload shape, phone normalization, profile gating, upstream errors
- `src/app/api/callback/__tests__/callback.test.ts` — endpoint happy/unhappy paths with mocked Prisma + Claude + Vapi
